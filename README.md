
**Лабораторная работа №5. Разработка SPA на React (интерфейс гостя).**

**Цель:** Создание базового интерфейса на React, подключенного к API.

**Задание:**
1.  Разработать 3 страницы на React + TypeScript + React-Bootstrap: Главная (статическое описание), Каталог расчетов, Детали расчета.
2.  На странице каталога реализовать все фильтры (по названию, точности данных) с отправкой параметров на бэкенд.
3.  Получать данные через `fetch` к вашему API. Решить проблему CORS через прокси в настройках React.
4.  Реализовать навигацию: `Navbar` и самописные `Breadcrumbs`.
5.  Для карточек предусмотреть изображение-заглушку, если `plot_url` пуст.
6.  Подготовить fallback: если бэкенд недоступен, данные берутся из mock-объектов.
7.  Обновить диаграмму классов, добавив компоненты фронтенда.
