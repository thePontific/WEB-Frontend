**Лабораторная работа №6. Управление состоянием и развертывание.**

**Цель:** Внедрение Redux, адаптивности, создание нативного приложения и деплой.

**Задание:**
1.  Внедрить Redux Toolkit для хранения состояния фильтров на странице каталога.
2.  Добавить адаптивный дизайн для 3 страниц: изменение количества колонок и размеров карточек на разных ширинах экрана.
3.  Настроить PWA (манифест, service worker). Развернуть фронтенд на GitHub Pages.
4.  Создать нативное приложение для гостя на Tauri (или другом фреймворке), включающее 3 страницы и подключающееся к API по IP локальной сети.
5.  Создать Deployment-диаграмму (StarUML) со всеми узлами системы: фронтенд, веб-сервис, БД, Redis, Minio.
